# B2B Каталог Мебельных Фабрик — Идеи Дизайна

## Выбранный подход: **Минималистичный Профессионализм с Индустриальными Акцентами**

### Design Movement
**Bauhaus + Contemporary B2B Minimalism** — функциональность встречается с эстетикой. Строгие линии, четкая типография, стальные и натуральные материалы как визуальные метафоры производства.

### Core Principles
1. **Функциональная ясность**: Каждый элемент служит цели. Нет декоративного шума — только информация, которая имеет значение.
2. **Доверие через прозрачность**: Открытая информация о фабриках, честные контакты, четкая иерархия данных.
3. **Производственная эстетика**: Визуальные отсылки к производству (металл, текстуры, геометрия) без китча.
4. **Масштабируемость**: Дизайн работает для 10 фабрик и для 100 — система остается стройной.

### Color Philosophy
- **Primary**: Темный сланец (`#2C3E50`) — серьезность, профессионализм, индустрия
- **Accent**: Теплый медный оттенок (`#D97706`) — энергия, производство, тепло человеческого контакта
- **Background**: Светлый серый (`#F8F9FA`) — чистота, минимализм, фокус на контенте
- **Secondary**: Светлый сланец (`#546E7A`) — мягкие переходы, вторичная информация
- **Reasoning**: Сланец + медь отражают материальность мебельного производства, светлый фон обеспечивает читаемость B2B контента.

### Layout Paradigm
- **Асимметричная сетка**: Левая колонка для фильтров/навигации (20%), основной контент справа (80%)
- **Карточки с минимальным отступом**: Каждая фабрика — компактная карточка с четкой иерархией (название → город → специализация → контакты)
- **Горизонтальные разделители**: Вместо вертикальных колонок, используем горизонтальные секции для России/Беларуси
- **Избегаем центрирования**: Контент выравнивается слева, создавая ощущение движения вверх/вниз по странице

### Signature Elements
1. **Геометрические разделители**: Тонкие горизонтальные линии с медным акцентом между секциями
2. **Иконки производства**: Небольшие SVG-иконки (станок, чертеж, кухня, шкаф) рядом с категориями
3. **Таблица как главный элемент**: Не скрытая в карточках, а визуально доминирующая, с мягкой тенью и медным заголовком

### Interaction Philosophy
- **Hover-эффекты**: Карточка фабрики получает мягкую тень и медный левый бордюр (не полная трансформация)
- **Фильтры как переключатели**: Клики на фильтры не перезагружают страницу, контент обновляется плавно
- **Копирование контактов**: Клик на телефон/email копирует в буфер обмена с тостом подтверждения
- **Нет агрессивных CTA**: Кнопки "Связаться" мягкие, вторичные — фокус на информации

### Animation
- **Entrance**: Карточки фабрик появляются с небольшой задержкой (stagger 30ms), opacity 0→1, translateY: 8px→0
- **Hover**: Левый бордюр растет с 2px до 4px, тень усиливается (200ms ease-out)
- **Filter transitions**: Контент переходит между состояниями за 250ms с ease-in-out
- **Micro-interactions**: Клик на кнопку копирования — быстрое масштабирование (0.97→1) за 100ms
- **Respect prefers-reduced-motion**: Все анимации отключаются для пользователей с этим предпочтением

### Typography System
- **Display (заголовки)**: **IBM Plex Serif** (700, 48px) — серьезность, индустриальность
- **Body (основной текст)**: **IBM Plex Sans** (400, 16px) — читаемость, нейтральность
- **Accent (категории, фильтры)**: **IBM Plex Sans** (600, 14px) — выделение без крика
- **Hierarchy**: H1 (48px) → H2 (32px) → H3 (24px) → Body (16px) → Small (14px)
- **Line-height**: 1.6 для body, 1.2 для заголовков — дышащий, не тесный текст

### Visual Assets
- **Hero background**: Абстрактное изображение производственного цеха (минималистичное, с медными акцентами)
- **Section dividers**: Тонкие медные линии с геометрическими узорами
- **Иконки**: Простые, монолинейные, в стиле Feather Icons, медного цвета

---

## Отклоненные подходы

### Подход 1: Яркий Стартап-дизайн
- **Проблема**: Слишком молодежный для B2B аудитории, которая ценит серьезность и надежность.

### Подход 2: Корпоративный Синий
- **Проблема**: Скучный, безличный, не отражает производственную природу бизнеса.

## Реализация

Все эти принципы будут воплощены в:
- Tailwind CSS с кастомными переменными (сланец, медь, серый)
- React компоненты с минимальными анимациями
- Таблица как основной компонент, а не вспомогательный элемент
- Фильтры, которые работают без перезагрузки
- Контакты, которые легко копировать
